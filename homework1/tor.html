Tor

     Tor is a free open source browser that protects your privacy and anonymity while surfing the web. Using Tor is similar to using any other popular web browser such as Chrome, Safari, or Firefox. However the difference between using these browsers and Tor is that Tor redirects all of your network traffic through a series of encrypted relays before putting you in touch with your end host. This helps maintain your privacy and anonymnity by making it significantly harder for outside attackers to discover the source of your network traffic. Tor can also be used to bypass certain forms of internet censorship. For example, since Tor hides the source of internet traffic, if a certain website is blocked by a university or local government, the Tor network could be used to make it appear as if the request was coming from another location, providing the user access to blocked website. It should be mentioned that since all your internet traffic is relayed multiple times when using Tor, web browsing when using Tor can be a little slower when compared to other browsers. For this reason, Tor likely won't replace your default browser. However, whenever you need private and anonymous browsing, you should definitely be using Tor.
